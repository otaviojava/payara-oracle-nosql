Getting Started

Quick Start with Docker Compose

The easiest way to run this application:

# Clone and navigate to the project cd payara-oracle-nosql # Start both Payara and Oracle NoSQL docker-compose up -d # View logs docker-compose logs -f # Access the application open http://localhost:8080

Local Development

For local development with hot reload:

# Start Oracle NoSQL database docker run -d --name oracle-nosql -p 9999:8888 \ ghcr.io/oracle/nosql:latest-ce # Run application with Payara Micro mvn clean package payara-micro:dev

API Endpoints

The application exposes the following REST endpoints:

GET /api/books - List all books (with pagination)
GET /api/books/{id} - Get book by ID
POST /api/books - Create new book
PUT /api/books/{id} - Update existing book
DELETE /api/books/{id} - Delete book

Example API Usage

# Create a new book curl -X POST http://localhost:8080/api/books \ -H "Content-Type: application/json" \ -d '{ "title": "The Great Gatsby", "author": "F. Scott Fitzgerald", "genre": "FICTION", "publicationYear": 1925, "tags": ["Classic", "American Literature"] }' # Get all books curl http://localhost:8080/api/books?page=1&size=10
